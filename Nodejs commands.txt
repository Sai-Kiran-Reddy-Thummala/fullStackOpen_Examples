NPM - node package manager

node index.js - executes the file index.js (similar can be done with npm by writing a script for it in package.json file ex: "start" : node index.js)

node init - initializes a new project

npm run dev - starts the development sever

npm --watch index.js - works like hot module replacement similar to vite hmr

npm install json-server --save -dev - installs the json-server and saves it in development dependencies

npm run dev - this runs the code in development mode, to this adding run is important
